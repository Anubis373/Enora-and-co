# Equipe Enora&co
## Comment accéder au jeu snake caché
Pour accéder au jeu du snake, il faut dévérouiller l'amélioration qui coute 50 points dans la boutique du clicker

## Comment accéder au jeu LaserGame
Pour accéder au laser game, il faut aller tout en bas de la page, puis de cliquer sur le bouton "Aller au combat"

## Comment accéder au formulaire
Cliquer sur le bouton "Remplire le formulaire"
